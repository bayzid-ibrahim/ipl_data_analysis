# 🏏 IPL Batting Evolution — Data Analysis

A complete data analysis of how **run-scoring has evolved in the IPL** from 2008 to 2026,
how batsmen became the dominant force, and where the historical turning points were.

## 📊 Dataset Overview

| Property | Value |
|----------|-------|
| Source | [Cricsheet](https://cricsheet.org/) IPL JSON archive |
| Coverage | All IPL matches up to **April 28, 2026** |
| Rows | 287,353 (one per legal delivery) |
| Columns | 28 |
| Innings covered | 1,575+ |

Each row represents the **state of an innings after one legal delivery** — runs scored so far,
wickets lost, batter stats, bowler stats, and the final innings total.

> **Note:** Wides and no-balls are not counted as legal deliveries. They do not increment `balls_bowled`.

### Key Columns

| Column | Description |
|--------|-------------|
| `innings` | 1st or 2nd innings |
| `balls_bowled` | Legal balls bowled so far |
| `balls_remaining` | Balls remaining in innings |
| `runs_scored` | Total runs scored so far |
| `wickets_lost` | Wickets fallen so far |
| `current_run_rate` | Cumulative run rate at this point |
| `striker_strike_rate` | Running strike rate of the on-strike batter |
| `striker_batting_position` | Batting order position of the striker |
| `bowler_economy` | Running economy rate of the current bowler |
| `innings_phase` | 1=Powerplay (1–36 balls), 2=Middle (37–90), 3=Death (91–120) |
| `target_final_score` | The final innings total (what we analyse) |

### Section 3 — Extract One Row Per Innings

Bucket all innings totals into 20-run bands (`<120`, `120–140` ... `220+`) to see

## 🕰️ Historical Turning Points

| Era | Key Shift |
|-----|-----------|
| **2008–2010** | Conservative era — avg scores ~145–155, bowling dominates |
| **2011–2012** | **Gayle's 175*** recalibrates what's possible — 160 becomes the norm |
| **2013–2016** | Death-over specialists emerge (Dhoni, AB, Pollard) — 170+ becomes common |
| **2017–2019** | 200 is routine — middle-order power-hitters reshape team selections |
| **2020–2022** | Suryakumar era; sustained high scoring even in neutral venues |
| **2023–2026** | **Impact Player rule** gives teams 7 batting options — bowling further diluted |

## 🛠 Notes & Assumptions

- Only **legal deliveries** are counted (`balls_bowled` excludes wides and no-balls).
- Strike rate analysis uses a minimum of **10 balls faced** to reduce noise.
- Economy rate analysis uses a minimum of **6 balls bowled** per bowler snapshot.
- Innings-level aggregation uses `["venue_id", "city_id", "stage_id", "league_match_number", "innings"]`
  as a composite key to identify unique innings (since there is no explicit match ID column).
- The dataset does **not** include player names or season/year — it is designed for ML score prediction.
  Historical narrative is derived from the aggregate patterns combined with domain knowledge.
